Import Instructions

  • Client Profiles: Excel/CSV file with columns: Name, Email, Phone, Company, Goals, Pain Points, Budget, Timeline, etc.
  • Meetings: iCal files from Google Calendar/Outlook, or CSV with Date, Time, Client, Type, Location
  • Backup Restore: Only use backup files generated by this platform
  • File Size: Maximum 10MB per file
  • Data Validation: All imported data will be validated before processing
